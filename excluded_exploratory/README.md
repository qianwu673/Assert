# Excluded Exploratory Runs

This directory contains exploratory rows that are retained for transparency but are not used for the reported primary quantitative claims.

`openai_gpt_5_mini_annotation_rows_excluded_from_main.csv` contains the 90 GPT-5-mini annotation rows extracted from the adjudicated all-results archive. GPT-5-mini is excluded from the main analysis because it required model-specific request settings, including omitted temperature and reasoning/output-token adjustments.
